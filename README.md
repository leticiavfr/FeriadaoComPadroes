# Sistema de E-commerce — Aplicação de Padrões de Projeto

## Visão Geral

Este projeto implementa um **sistema de e-commerce** com foco na aplicação de **padrões de projeto (Design Patterns)** para garantir:

* Baixo acoplamento
* Alta coesão
* Facilidade de manutenção
* Facilidade de extensão

O sistema simula o fluxo de compra de um usuário, incluindo criação de pedidos, aplicação de descontos, processamento de pagamentos e envio de notificações.

---

## Objetivo

Demonstrar, de forma prática, como diferentes padrões de projeto podem ser combinados para resolver problemas reais de engenharia de software.

---

## Funcionalidades

* Criação de pedidos
* Aplicação de cupons e adicionais
* Processamento de pagamentos (Pix, cartão, boleto)
* Envio de notificações ao cliente
* Controle de tentativas e validações
* Interface simplificada para o usuário

---

## Conceitos Demonstrados

* Desacoplamento entre módulos
* Extensibilidade (adicionar novos pagamentos ou notificações facilmente)
* Separação de responsabilidades
* Reutilização de código

---

## Conclusão

Este projeto demonstra como a aplicação correta de padrões de projeto pode transformar um sistema complexo em uma solução organizada, escalável e de fácil manutenção.

---

## Autor
